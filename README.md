# 文档水印工具 v1.2

MasterLin 开发的 Windows 文档水印工具。

## 功能

- 支持 Word `.docx` 和 PDF `.pdf` 添加水印
- 支持图片上方水印
- 支持拖拽添加源文件
- 支持明暗主题自动切换
- 支持创建桌面快捷方式

## 使用

直接运行 `文档水印工具_v1.2.exe`。

注意：程序运行时需要同目录下的 `assets` 文件夹，以及 PDF/压缩相关 DLL 依赖。加水印后的文档主要用于分发/打印，可能不适合继续编辑，请保留源文件。

## 版本规则

版本号采用两段式：`主版本.维护版本`。

维护更新递增后一位，例如 `1.2 -> 1.3`；重大功能升级递增主版本，例如 `1.9 -> 2.0`。
